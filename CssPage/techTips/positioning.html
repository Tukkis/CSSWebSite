Flexbox

Flexbox is a CSS layout module that allows you to change the element flow in the DOM from vertical to horizontal inside the element that is given the display:flex property and value.
Flexing elements is a very quick and dependable way to build your layouts.

Flexbox can only be used to create one-dimensional layouts, however when nesting flexboxes inside each other you can do pretty much anything with flex.

The syntax for flexbox is very straightforward i urge you to visit css-tricks article on flexbox for a good guide on all the properties and values for disply:flex.

Grid

CSS Grid is a display property that allows you to divide the page into regions of specified sizes and locations. To use grid give the parent element the propery value paid display:grid
Using grid allows you to very thoroughly place you elements in any way you please. It may not be the quickest way but it is dependable.

Grids may act very oddly at first glance, but once you get to know them, you may find the love of your life.

Grid syntax is straightforward and has many similarities with the syntax of flexbox. css-tricks has a good article on grid as well.

Float

The float property allows you to "float" elements on the right or the left of other elements in the DOM. Give element float the property with float:(right/left).
The most common use for float is wrapping text around images.

Float is, in most cases, inferior to the other positioning techinques. It often causes unexpected interactions with other elements and can be a real nuisance.
