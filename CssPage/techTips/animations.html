Keyframes

Animations in CSS use keyframes, meaning you specify the start and end states, and as many states in between as you want, as keyframes. The browser will then animate the transitory frames.

Keyframes have a similar syntax to media queries. @keyframes animationname {keyframes-selector {css-styles;}}.

Animation duration

The animation duration property defines how long it takes the engine to complate the animation from the first keyframe to the last.

Takes in value as seconds (s).

Animation delay

Animation delay specifies the delay for the animations start.

Animation delay takes in value as seconds (s) and can take negative values, in which case the animation will play as if it had been playing for that amount of seconds.

Iteration count

Iteration count is a property that defines how many times the animation will be played.

If you want the animation to be played indefinetly the iteration count property can take the value of infinite.
