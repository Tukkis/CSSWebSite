Tailwind

Tailwind.CSS is a CSS framework that is created to help users write and maintain website faster and easier.
This framework is very light, meaning it is very powerfull for small projects, but doesn't scale well for very large ones.

The structure in tailwind differs from vanilla CSS by having everything you need packed in prebuilt classes.
You dont need to write custom CSS to give an element a padding or a border. Instead to give a padding you give the element the calss p with the size of the padding you want.
Super easy!

To use tailwind in your own projects go to the official tailwindcss website and follow their documentation.

Bootstrap

Bootstrap is a open source CSS framework that is designed to enable responsive developement for mobile-first websites.
Being easy to learn and master Bootstrap is a usefull framework to get your bearings with.
More than 19% of websites use bootstrap!
Bootstrap works on a grid system, where you create your layout using rows and columns. This causes making responsive websites to be a breeze!

To use bootstrat in your own projects go to the official bootstrapcss webiste and follow their documentation.
